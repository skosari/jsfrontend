Scope!

  1. var - (global) - var will declare a variable that will have scope outside of a function. So if function one() declares var myVar = "string1", function two(string1) will work because var can jump from inside function one into function two.
  2. let - (local) - let will only work inside the function it will be declared in. For recursive functions let will not work because the function being called by the main base function will not carry over
  3. const - (global) - we use const when we do not want our variable to change types and for our variable to be available globally
